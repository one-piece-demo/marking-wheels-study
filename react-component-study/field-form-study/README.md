# field-form-study

> 简易实现[react-component/field-form](https://github.com/react-component/field-form)

## Getting Started

Install dependencies,

```bash
npm i
```

Start the dev server,

```bash
npm start
```

Build documentation,

```bash
npm run docs:build
```

Build library via [father-build](https://github.com/umijs/father),

```bash
npm run build
```
